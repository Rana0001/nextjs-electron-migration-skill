# Example Prompts

## 1) Full migration plan

Convert my Next.js app to a production-ready Electron desktop app.
Use the nextjs-electron-migration skill and provide:

- architecture decisions
- migration steps
- security hardening
- packaging strategy
- CI plan
- risk mitigation

## 2) Security-focused

Review my Electron + Next integration and harden it using best practices.
Focus on:

- BrowserWindow security settings
- preload bridge restrictions
- IPC validation
- navigation allowlisting

## 3) CI and release

Create a cross-platform CI strategy for Electron packaging with smoke tests
for macOS, Windows, and Linux.

## 4) Troubleshooting

My packaged Electron app shows a blank screen.
Diagnose likely root causes and give a prioritized fix plan.
